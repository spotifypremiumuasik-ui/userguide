# Menu: Dori Proses

## I. Buat Penerimaan Baru
<a id="i-buat-penerimaan-baru"></a>

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Ikuti langkah-langkah berikut:

1. **Buka Aplikasi:** Buka aplikasi melalui browser (disarankan **Google Chrome**), lalu akses:  
   [https://synergis.berkat-agung.com/tallyikan/product-index.php](https://synergis.berkat-agung.com/tallyikan/product-index.php)

2. **Login:** Masukan **Username** dan **Password**, lalu klik **Login**.

   ![Halaman Login](images/TallyProses/halamanLogin.png)
   *Gambar I.1: Halaman Login.*

3. **Masuk Menu Dori:** Pilih menu **Dori**.

   ![Menu Dori](images/dori/sesi1.png)
   *Gambar I.2: Menu Dori.*

4. **Masuk Menu Pendaftaran:** Pilih menu **Pendaftaran** untuk membuat penerimaan baru.

   ![Menu Pendaftaran](images/dori/sesi2.png)
   *Gambar I.3: Menu Pendaftaran.*

5. **Masukan Data:** Isi data berikut:
   - **Tally ID**
   - **Tanggal Timbang**
   - **Pemasok**
   - **Nomor Kendaraan**

   ![Form Pendaftaran](images/dori/sesi3.png)
   *Gambar I.4: Form Pendaftaran.*

6. **Simpan Data:** Setelah selesai, klik tombol **Simpan** untuk menyimpan data penerimaan.

---

## II. Masukan Timbangan
<a id="ii-masukan-timbangan"></a>

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Penimbangan dilakukan **dua kali** untuk memastikan keakuratan data. Ikuti langkah-langkah berikut:

1. **Masuk ke Daftar Tally:** Buka menu **Daftar Tally**.

   ![Daftar Tally](images/dori/timbangan1.png)
   *Gambar II.1: Daftar Tally.*

2. **Pilih Tally:** Temukan dan pilih **Tally** yang sudah dibuat.

   ![Pilih Tally](images/dori/timbangan2.png)
   *Gambar II.2: Pilih Tally.*

3. **Masuk Menu RECV:** Klik menu **RECV** untuk melakukan penimbangan awal.

   ![Menu RECV](images/dori/timbangan3.png)
   *Gambar II.3: Menu RECV.*

4. **Masukan Data:** Isi data berikut:
   - **Nama Produk**
   - **Potongan Nota**
   - **Catatan**

   ![Input Data RECV](images/dori/timbangan4.png)
   *Gambar II.4: Input Data RECV.*

5. **Mulai Timbang:** Klik **Mulai Timbang**, lalu lakukan penimbangan.  
   Data akan tersimpan dalam bentuk **card**.

   ![Proses Timbang](images/dori/timbangan5.png)
   *Gambar II.5: Proses Timbang.*

6. **Timbangan Kedua (Validasi):**

   a. **Masuk Menu GW:** Buka menu **GW**.

   ![Menu GW](images/dori/timbangan6.png)
   *Gambar II.6: Menu GW.*

   b. **Pilih Produk:** Pilih produk yang akan ditimbang ulang.

   ![Pilih Produk](images/dori/timbangan7.png)
   *Gambar II.7: Pilih Produk.*

   c. **Mulai Timbang:** Lakukan penimbangan kedua sebagai validasi.

   ![Timbang Kedua](images/dori/timbangan8.png)
   *Gambar II.8: Timbangan Kedua.*

---

## III. Masukan Proses
<a id="iii-masukan-proses"></a>

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Tahapan ini mencakup seluruh proses pengolahan setelah penimbangan. Ikuti langkah-langkah berikut:

1. **Pra Proses (PRE):**

   a. Masuk ke menu **PRE**.

   ![Menu PRE](images/dori/proses1.png)
   *Gambar III.1: Menu PRE.*

   b. Klik **Mulai Timbang** untuk melakukan penimbangan sebelum proses.

   ![Mulai Timbang PRE](images/dori/proses2.png)
   *Gambar III.2: Mulai Timbang PRE.*


2. **Proses (PRO):**

   a. Masuk ke menu **PRO**.

   ![Menu PRO](images/dori/proses3.png)
   *Gambar III.3: Menu PRO.*

   b. Masukan **Pelaksana Proses**.

   ![Pelaksana Proses](images/dori/proses4.png)
   *Gambar III.4: Pelaksana Proses.*

   c. Masukan **Nama** dan lakukan **Checklist Bagian**.

   ![Checklist Bagian](images/dori/proses5.png)
   *Gambar III.5: Checklist Bagian.*

   d. Klik **Daftarkan** untuk menyimpan.


3. **Masukan Hasil Timbang:**

   a. Masuk ke menu **Timbang**.

   ![Menu Timbang](images/dori/proses6.png)
   *Gambar III.6: Menu Timbang.*

   b. Pilih bagian yang ingin di timbang.

   ![Input Timbangan Proses](images/dori/proses7.png)
   *Gambar III.7: Pilih Bagian.*

   c. lakukan penimbangan.

   ![Input Timbangan Proses](images/dori/proses8.png)
   *Gambar III.8: Input Timbangan Proses.*

   d. Ulangi sampai semua bagian terpenuhi


4. **Soaking (SOA):**

   a. Masuk ke menu **SOA**.

   ![Menu SOA](images/dori/proses9.png)
   *Gambar III.9: Menu SOA.*

   b. Daftarkan **Mesin** yang digunakan.

   ![Daftar Mesin](images/dori/proses10.png)
   *Gambar III.10: Daftar Mesin.*

   c. Pilih **BL**.

   ![Pilih BL](images/dori/proses11.png)
   *Gambar III.11: Pilih BL.*

   d. Masukan **Berat Awal** dan **Lama Proses Soaking**, lalu klik tombol **+** untuk setiap input.

   ![Input Soaking Awal](images/dori/proses12.png)
   *Gambar III.12: Input Soaking Awal.*

   e. Klik **Simpan** untuk menyimpan hasil timbangan awal.

   ![Simpan Awal](images/dori/proses13.png)
   *Gambar III.13: Simpan Awal.*

   f. Klik **Edit**, lalu masukan **Berat Akhir** dan klik tombol **+** untuk setiap input.

   ![Input Soaking Akhir](images/dori/proses14.png)
   *Gambar III.14: Input Soaking Akhir.*

   g. Klik **Simpan** untuk menyimpan hasil akhir.

   ![Simpan Akhir](images/dori/proses15.png)
   *Gambar III.15: Simpan Akhir.*

    Maka hasilnya akan seperti berikut:

   ![Simpan Akhir](images/dori/proses16.png)
   *Gambar III.16: Simpan Akhir.*

   h. Lakukan langkah yang sama untuk produk **NBL**.


5. **Freezing (FRZ):**

   a. Masuk ke menu **FRZ**.

   ![Menu FRZ](images/dori/proses17.png)
   *Gambar III.17: Menu FRZ.*

   b. Pilih jenis produk **BL** 

   ![Pilih FRZ](images/dori/proses18.png)
   *Gambar III.18: Pilih FRZ.*

   c. Timbang **Berat Awal** dan **Berat Akhir**, lalu klik **Simpan**.

   ![Input FRZ](images/dori/proses19.png)
   *Gambar III.19: Input FRZ.*

   d. Ulangi langkah untuk **NBL**.

6. **Glazing (GLZ):**

   a. Masuk ke menu **GLZ**.

   ![Menu GLZ](images/dori/proses20.png)
   *Gambar III.20: Menu GLZ.*

   b. Daftarkan seluruh produk yang akan diproses, klik **Tambah Produk**

   ![Daftar Produk GLZ](images/dori/proses21.png)
   *Gambar III.21: Daftar Produk GLZ.*

    c. Masukan nama produk dan klik daftarkan.

   ![Menu Timbang GLZ](images/dori/proses22.png)
   *Gambar III.22: Masukan Nama Produk.*

   d. Masuk ke menu **Timbang**.

   ![Menu Timbang GLZ](images/dori/proses23.png)
   *Gambar III.23: Menu Timbang GLZ.*

   e. Pilih tahap **Sebelum** dan lakukan penimbangan, lalu simpan.

   ![Sebelum Glazing](images/dori/proses24.png)
   *Gambar III.24: Sebelum Glazing.*

    f. Lakukan penimbangan lalu klik **Simpan**.

   ![Sebelum Glazing](images/dori/proses25.png)
   *Gambar III.25: Sebelum Glazing.*

   g. Lakukan langkah yang sama untuk **Sesudah Glazing** dan **MCO Lama** (jika ada).

## IV. Packing
<a id="iv-packing"></a>

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Tahapan terakhir adalah proses **Packing**. Ikuti langkah-langkah berikut:

1. **Scan Packing:**

   a. Masuk ke menu **SCAN**.

   ![Menu SCAN](images/dori/packing1.png)
   *Gambar IV.1: Menu SCAN.*

   b. Pilih menu **Packing**.

   ![Pilih Packing](images/dori/packing2.png)
   *Gambar IV.2: Pilih Packing.*

   c. Lakukan **Scan** untuk setiap hasil packing.

   ![Proses Scan](images/dori/packing3.png)
   *Gambar IV.3: Proses Scan.*

2. **Packing (PCK):**

   a. Masuk ke menu **PCK**.

   ![Menu PCK](images/dori/packing4.png)
   *Gambar IV.4: Menu PCK.*

   b. Sesuaikan **Berat (Kg) per MC** untuk semua produk.

   ![Setting Berat](images/dori/packing5.png)
   *Gambar IV.5: Setting Berat.*

   c. Klik **Simpan** untuk menyimpan hasil akhir packing.
